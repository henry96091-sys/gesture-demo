4. 使用方式

  1. 兩台裝置都用瀏覽器打開這個頁面。
  2. 每台都會自動拿到自己的 Peer ID。
  3. 把 A 裝置的 Peer ID 複製到 B，或反過來都可以。
  4. 按「連線」成功後，再按「開啟相機」。
  5. 鏡頭辨識到手勢時，會把手勢名稱送到另一台。
